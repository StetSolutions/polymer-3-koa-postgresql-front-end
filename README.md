# Simple Front-end

## Synopsis

Simple front-end using [Polymer App Toolbox - Starter Kit](https://github.com/Polymer/polymer-starter-kit)

## Install

Open terminal at project root and execute

    $ npm install

## Serve

    $ polymer serve
